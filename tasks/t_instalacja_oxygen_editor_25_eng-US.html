Oxygen Editor 25 installation

Installing Oxygen XML Editor on Windows is done by following on-screen instructions through a dedicated installation wizard.

Download the installer suitable for your computer, and then proceed with the installation:

  • download for Microsoft Windows
  • Download for Mac OS
  • Download for Linux
  1. Double-click the previously downloaded file named oxygen-64bit-openjdk.exe and wait for the installation wizard to begin.

    By default, all downloaded files are located in the Downloads folder inside the Windows user directory

  2. In the (Language selection) dialog box, make sure English is set and click OK.
  3. In the (User Account Control) dialog, click Yes to allow the Oxygen XML Editor installer to make changes on your computer.
  4. Follow the on-screen instructions and confirm your choices with the Next button.
  5. At the (Select File Associations) stage, you can leave the default associations or optionally customize the selection to your liking. Click Next and complete the installation.
  6. In the final installation step, make sure that Run Oxygen XML Editor option is checked and click Finish.
Oxygen XML Editor 25 has been installed successfully.

Restart the computer.
